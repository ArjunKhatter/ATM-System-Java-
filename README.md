# 🏦 ATM System (Java)

## 📌 Description

A simple ATM system built using Java that allows users to perform basic banking operations like checking balance, depositing money, and withdrawing money using a PIN-based login.

---

## 🚀 Features

* PIN Authentication
* Check Account Balance
* Deposit Money
* Withdraw Money
* Menu-driven interface

---

## 🛠️ Tech Stack

* Java
* Scanner Class (for input)

---

## 🔑 Default PIN

```
1234
```

```

```

## 💡 Concepts Used
